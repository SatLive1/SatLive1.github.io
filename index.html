一、MEO与LEO卫星通信协同已落地成功案例

1. SES与SpaceX星链的MEO-LEO邮轮通信协同服务

该案例为商业化成功典范：2023年9月，SES公司与SpaceX达成合作，集成SES的MEO卫星服务与星链的LEO卫星服务，推出邮轮行业首个MEO-LEO融合连接方案。

  • 核心优势：结合MEO卫星的稳定覆盖与LEO卫星的低时延特性，实现全天候高速安全网络连接；
  • 性能指标：最高提供3Gbps传输速度；
  • 解决痛点：彻底解决远洋邮轮通信不稳定、带宽不足问题，满足游客在线支付、社交媒体访问及船舶管理数据传输等需求。

2. SpaceLink的MEO中继网络与LEO卫星的协同技术验证

该案例完成多项关键场景协同验证，属技术层面成功实践：SpaceLink构建的MEO中继网络计划部署4颗MEO卫星，配备光学和射频通信链路，突破传统LEO卫星仅能在地面站上空传输数据的限制。

  • 与美国陆军达成五年协同研发协议：通过仿真和模型验证，该网络可缩短传感器到作战人员的通信延迟，快速传输卫星图像等数据；
  • 获NASA合同：在莫哈韦沙漠网关站完成与NASA近地网络的光学地面终端集成及安全数据中继演示；
  • 入选DARPA的Space-BACN项目：参与研发商业通信星座与国防系统的交互协议，奠定军事领域MEO-LEO协同通信技术基础。

3. 中国移动“卫星 + 模组”的天地协同通信实践

国内商业化场景成功案例，结合低轨与中高轨卫星协同潜力实现落地：

  • 2024年发射的“中国移动01星”（搭载5G天地一体演进技术星载基站）可与国产LEO卫星协同，2025年推出的MU305系列卫星模组适配该卫星，地面信号弱时0.3秒内自动切换至LEO卫星链路，海洋/雪山等无地面网络区域通信稳定；
  • 2023年中国移动香港联合垣信卫星等，在“鼓浪屿号”邮轮上成功测试国产低轨“千帆星座”，该低轨网络借助中轨卫星中继拓展覆盖，实现邮轮高速网联服务，完成低轨与中高轨卫星协同支撑海事通信的阶段性验证。

二、MEO与LEO卫星通信协同其他实践案例

1. SpaceLink公司MEO中继网络（前身Audacy）

2018年Audacy提出用3颗MEO卫星为LEO卫星提供数据中继服务并获FCC频率许可，2020年重组为SpaceLink后持续推进；其MEO中继系统通过光学/射频链路实现LEO卫星数据即时转发，大幅提升传输连续性和效率。

2. SES公司跨星座协同

SES（运营GEO/MEO卫星的国际通信巨头）与星链、OneWeb等LEO星座合作，打造多轨道无缝通信网络：MEO卫星凭借“少量卫星覆盖全球”优势，分担LEO星座通信压力，通过星间链路实现LEO卫星数据中继与调度，提升网络覆盖稳定性和通信容量。

3. 中国天地一体化信息网络探索

我国“天地一体化信息网络”规划中，MEO卫星作为天基骨干网核心节点，辅助LEO卫星数据中继与管理：分担路由表计算压力，通过星间链路实现LEO数据高效传输，支撑航空航天、遥感观测等领域通信需求。

4. 中国NTN制式协同试验（2025）

中国电信研究院等依托“智慧天网01星”（2万公里中轨卫星）完成NTN制式在轨验证，后续可与LEO星座协同：MEO保障大范围基础通信链路，LEO聚焦低时延高速传输，支撑极地科考、远洋通信等无地面网络场景。

三、核心技术术语英译

强化学习路由算法：Reinforcement Learning Routing Algorithm（缩写：RL Routing Algorithm）

  • 补充：若强调“基于强化学习的”，可表述为 Reinforcement Learning-based Routing Algorithm（适用于学术标题/摘要）；
  • 应用示例：This paper proposes an RL Routing Algorithm for satellite networks.（本文提出一种面向卫星网络的强化学习路由算法）。
