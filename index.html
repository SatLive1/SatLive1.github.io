SatLive

SatLive is an end-to-end live video streaming framework designed for Integrated Satellite-Terrestrial Networks (ISTNs). It adopts a hierarchical reinforcement learning (RL)-driven dynamic routing solution, adapting to the rapid topology changes of Low Earth Orbit (LEO) satellites and prioritizing video Quality of Experience (QoE). Through mechanisms like cross-domain routing and video-aware transmission, SatLive reduces average packet delay by at least 28.8% and improves QoE by 1.13x. The source code and dataset have been open-sourced and are available at: GitHub

README

Overview

This project implements a satellite routing system supporting loop detection and intelligent edge node selection. Specifically optimized for MEO-LEO constellation routing scenarios, the system integrates a variety of advanced routing strategies and performance monitoring functionalities.
